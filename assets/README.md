# Assets - Memorial Maker

## Logo TecPred

Coloque o arquivo `logo_tecpred.png` neste diretório.

O logo será usado automaticamente em todos os memoriais gerados.

**Formatos aceitos:**
- PNG (recomendado)
- JPG/JPEG
- Tamanho sugerido: 400x100px ou similar

**Localização no documento:**
- Cabeçalho do memorial
- Primeira página
